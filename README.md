# Backdoor-Detector-Auditor
This tool audits your Linux system. It detects suspicious network connections, background processes, and unauthorized users.
Enter this command to give the script executable permission:
```Bash
chmod +x backdoor_detector.sh
```
Here's the tool, run it using sudo:
```Bash
sudo ./backdoor_detector.sh
```
(Note: Here you will be asked for your Linux system password, type it and press Enter. The password will not appear on the screen while you are typing, but it is being typed).
